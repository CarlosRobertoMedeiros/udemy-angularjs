Keyboard Events ng-keydown, ng-keypress, ng-keyup

Keyboard events are basically triggered when the keys are pressed or released

  • ng-keydown

    is triggered when the key is pressed and it is on the down position

  • ng-keypress

    is triggered when the key is pressed. It does not recognise CTRL, ALT or SHIT

  • ng-keyup

    it triggered when the key is released
